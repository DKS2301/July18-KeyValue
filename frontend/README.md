# XPressMeal Frontend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. (Optional) Configure API base URLs if backend is not on localhost:5000.

## Run

- Development:
  ```bash
  npm run dev
  ```
- Production:
  ```bash
  npm run build
  npm start
  ```

## Pages

- `/` — Landing page
- `/login` — Student login
- `/menu` — Menu & order
- `/success` — Order success
- `/admin` — Admin dashboard
